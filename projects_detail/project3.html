potentiometer

A speedo meter project with using esp32,potentiometer and a i2c oled diplay

Components
ComponentPurpose
ESP32×1Main microcontroller
I2C oled×1to display output
potentiometer×1for veriable valu
Jumper Wires×4Connections on breadboard
Breadboard×1Prototyping base
Wiring
  • VINESP32 3V3 → Oled VCC and one end of pot
  • GNDESP32 GND → Oled GND and other end of pot
  • CLKESP32 22 → Oled SCL
  • DATAESP32 21 → Oled SCA
  • potESP32 D4 → middile of pot
How to Build in IDE (Arduino)
  1. 1connectionconnect all component as shown and plug USB to PC.
  2. 2IDEtap the IDE button at the top of the web page.
  3. 3portclick request port and find cp2102 uart to bridge or similar to that.
  4. 4board and boudrate select select esp32 from list and boudrate to 115200
  5. 5project select potmeter from project list.
  6. 6passwarduse "dasiotpotmeter573" to unlock the project.
  7. 7uploadhit flash to upload
  8. 8upload errorif error occurs press boot button while flashing
  9. 9outputafter finishing upload press en or reset pin once
