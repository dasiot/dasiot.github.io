Smart Door Lock

A BLE and keypad-based smart door lock system built on ESP32. Users can unlock via a 4-digit PIN on a physical keypad or wirelessly through a Bluetooth mobile app. A servo motor controls the deadbolt mechanism.

Components
ComponentPurpose
ESP32 DevKit×1Main controller with BLE + WiFi
4×4 Keypad×1PIN entry for unlock
SG90 Servo Motor×1Controls deadbolt mechanism
LCD 16×2 (I2C)×1Status display (locked/unlocked)
5V Power Supply×1Powers ESP32 + servo
Buzzer×1Wrong PIN alert
Jumper WiresConnections
Wiring
  • GPIO 13ESP32 → Servo signal wire (orange)
  • GPIO 12-19ESP32 → Keypad rows & columns (R1–R4, C1–C4)
  • SDA/SCLESP32 GPIO 21/22 → LCD I2C SDA/SCL
  • GPIO 25ESP32 → Buzzer positive pin
  • VIN5V → Servo VCC & ESP32 VIN
  • GNDCommon GND for all components
How to Build in IDE (Arduino)
  1. 1Install Arduino IDEDownload from arduino.cc. Open File → Preferences and add the ESP32 board URL:
    https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
  2. 2Install ESP32 BoardGo to Tools → Board → Board Manager, search esp32 and install the Espressif package.
  3. 3Install Required LibrariesIn Sketch → Include Library → Manage Libraries install:
    Keypad by Mark Stanley, LiquidCrystal_I2C, and ESP32Servo. BLE is built-in.
  4. 4Set Your PINIn the code, set correctPIN = "1234" to your desired unlock PIN. Change serviceUUID if customizing the BLE service.
  5. 5Select Board & PortGo to Tools → Board and select ESP32 Dev Module. Select the correct port under Tools → Port.
  6. 6Upload the CodeHold the BOOT button on the ESP32 while clicking Upload if it does not start automatically. Wait for "Done uploading."
  7. 7Test Keypad & BLEEnter your PIN on the keypad — the servo should rotate to unlock. For BLE, install nRF Connect on your phone, find the device named SmartLock, and send 1234 to the characteristic to unlock remotely.
