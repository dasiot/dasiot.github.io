IoT Smart Light App

A smart light control application built using ESP8266. Control home lighting remotely over WiFi using the MQTT protocol, with support for scheduling and real-time status updates via a mobile dashboard.

Components
ComponentPurpose
ESP8266 NodeMCU×1Main WiFi microcontroller
5V Relay Module×2Switch AC bulbs on/off
Bulb + Holder×2Controlled light loads
5V Power Supply×1Power for ESP + relay
Jumper WiresConnections on breadboard
Breadboard×1Prototyping base
Wiring
  • D1ESP8266 D1 → Relay 1 IN signal pin
  • D2ESP8266 D2 → Relay 2 IN signal pin
  • VIN5V Power → Relay VCC & ESP VIN
  • GNDCommon GND for ESP + both relays
  • COMRelay COM → Live wire of AC circuit
  • NORelay NO → Bulb holder input
How to Build in IDE (Arduino)
  1. 1Install Arduino IDEDownload and install Arduino IDE from arduino.cc. Open it and go to File → Preferences.
  2. 2Add ESP8266 BoardIn Preferences, paste this URL in "Additional Board Manager URLs":
    http://arduino.esp8266.com/stable/package_esp8266com_index.json
    Then go to Tools → Board → Board Manager and install esp8266.
  3. 3Install Required LibrariesGo to Sketch → Include Library → Manage Libraries and install:
    PubSubClient (for MQTT) and ESP8266WiFi (built-in).
  4. 4Configure WiFi & MQTTIn the code, set your ssid, password, and mqtt_server variables to match your network and broker (e.g., broker.hivemq.com).
  5. 5Select Board & PortGo to Tools → Board and select NodeMCU 1.0 (ESP-12E). Then select the correct COM port under Tools → Port.
  6. 6Upload the CodeClick the Upload button (→). Wait for "Done uploading." Open Tools → Serial Monitor at 115200 baud to see connection status.
  7. 7Test with MQTT ClientUse an app like MQTT Dash or IoT MQTT Panel on your phone. Subscribe/publish to your topic (e.g., home/light1) to toggle the relay.
